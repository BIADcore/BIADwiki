MaterialCulture
Column name Data type Column description
MaterialCultureID varchar Starts with M000001
PhaseID varchar
ItemID varchar Link between datasets which integrate data of common origins.
Object varchar Functional category assigned in the primary literature. Currently a VARCHAR but in the future will be dependent on a zoptions table.
Material varchar Basic category of the material the object is made of.
MaterialType varchar A more detailed description of the resource used in the production of the object if provided in the primary literature.
TaxonCode varchar Unique code to identify the full taxonomic name extracted from zoptions_FaunalTaxaList (PROBLEM - SHOULD ALSO ALLOW THE USE OF BOTANICAL TAXA!)
Morphology varchar Basic morphological description of the object if available.
MorphologySection varchar Basic description of the object's morphology when observed in section e.g., two buttons could have a circular morphology when looked at from the top, but one could actually form a half-sphere and the other a cone when looked at from the side. Morphology would be 'circular' in both cases, but Morphology (section) would be 'half-sphere' in one case and 'conical' in the other. In the case of a blade, morphology in section could be 'oval', 'diamond-shaped', etc.
Preservation varchar Basic information about the state the artefact was reported in.
Decorated enum Binary distinction between objects with presence of decoration (1) and those without (0).
Retouched enum Binary distinction between objects with presence of retouche (1) and those without (0).
Polished enum Binary distinction between objects with polished surface (1) and those without (0).
Perforated enum Binary distinction between objects with presence of perforations (1) and those without (0).
Flat enum Binary distinction between objects with a flat shape (1) and those without (0).
DecorStyle varchar A more detailed description of the decorative style of the object if provided in the primary literature. See zoptions table.
DecorTechn varchar Basic information on the technique used to decorate the artefact. See zoptions table.
DecorTool1 varchar A more detailed description of the tool used to decorate the object if provided in the primary literature. See zoptions table.
DecorTool2 varchar A more detailed description of a second tool used to decorate the object if provided in the primary literature. See zoptions table.
DecorTool3 varchar A more detailed description of third tool used to decorate the object if provided in the primary literature. See zoptions table.
DecorToolTaxonCode varchar Unique code to identify the full taxonomic name for the species corresponding to the decoration tool (extracted from zoptions_TaxaList). This is mostly intended for pottery decorative styles created through the impression of mollusc shells (e.g., Cerastoderma edule, Cardium sp., etc.).
PerforationsNumber int Number of perforations through the object (original number of perforations if the object is fragmented but clearly presented additional perforations in the missing part (e.g., quadrangular plaque with perforations near each of the angles, but one angle is chipped so the perforation is no longer visible).
PerforationMorphology varchar A more detailed description of the perforations if provided in the primary literature, e.g., tubular, V-shaped (enters and resurfaces on the same plane), or hourglass-shaped.
Height/Length int Reported in mm. Usually the longest of the three dimensions; or in the case of pottery always the height (even if the pottery is wider than higher).
Width int Reported in mm.
Quantity int Number of objects with IDENTICAL characteristics.
CitationID varchar Unique identifier of a reference publication extracted from Citations table.
notes varchar
morphometrics enum Information whether an artefact was digitalized for morphometric analysis
