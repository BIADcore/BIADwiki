Strontium
Strontium isotopic data from human remains
Column name Data type Column description
StrontiumID smallint Unique primary key
IndividualID varchar Unique ID for each human specimen
ItemID varchar Must exist in the Items table.
Species varchar Must exist in the TaxonCode table
Material enum The material on which the analyses are based.
SkeletalElement varchar Skeletal element (case sensitive); lower case letters correspond to deciduous dentition and upper case correspond to adult dentition.
Leachate enum Whether strontium measurements were extracted from leachate from the sample element.
Side enum Side of with respect to the deceased, for elements that exist in symmetric pairs, such as teeth.
Jaw enum Teeth can be upper (maxillar) or lower (mandibular).
SegmentLength decimal Segment length (cm) amalgamates the isotopic signal of the months of growth within that sample. Hair grows circa 1 cm per month, so a 5 cm segment includes isotopic information for 5 months.
SegmentProxyTimeBeforeDeath varchar Segment distance from scalp/nailbed (cm). Hair grows circa 1 cm per month, therefore a segment that is 5 cm from the scalp informs on isotopic data 5 months before death.
SegmentMass decimal Mass of segment (mg).
Method varchar What analytical method e.g. laser ablation
LabCode varchar Original labcode provided by the lab that generated the strontium values.
Mean_87Sr/86Sr decimal Strontium isotopic ratio.
Error decimal Measurement error on the strontium ratio; equivalent to 2SE
ppm int
MassSpecType enum Type of Mass Spectrometry.
notes varchar
CitationID varchar
