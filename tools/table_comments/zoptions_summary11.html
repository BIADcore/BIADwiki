zoptions_FaunalMeasurement
Von den Driesch abbreviation of the measurement, and a description. WARNING: many measurement codes can be used for different elements such as GL (Greatest Length obviously is a property of any bone), however, some codes actually differ in their meaning for different elements. For example, FL/9 indicates a specific part of a mandible that is different to its meaning when applied to a cranium.
Column name Data type Column description
Measurement varchar Von den Driesch abbreviation of the measurement
Description varchar Full description of the Von den Driesch abbreviation
