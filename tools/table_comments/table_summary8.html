FaunalIsotopeSequences
Where sequential sampling data are available, raw δ13C(‰) and δ18O(‰) and potentially other sequential data should be stored here.
Column name Data type Column description
FaunSliceID varchar Primary key
FaunIsoID varchar
SampleID varchar Sample number or ID in the primary source specifying this particualar sequential sample
SequenceOrder tinyint Rank order of the sequence (i.e closest to the root is 1)
ERJdistance(mm) decimal Distance from Enamel Root Junction (mm)
δ13C_bioapatite(‰) decimal
δ18O_bioapatite(‰) decimal
CitationID varchar
