ABotPhases
Archaeobotanical data at the phase level, therefore only for phases that have this type of data. RecoveryMethod1, 2, 3 etc have no intrinsic differences, and the order is irrelevant. This is simply to allow for multiple recovery methods at a single phase.
Column name Data type Column description
PhaseID varchar Please see the detailed description of the 'Phases' table.
ABotNumberOfContexts smallint Number of distinct 'pits' or 'contexts' from which ABot data has been counted. Used to calculate unbiquity. Surface scatter would count as a single pit.
AbotMeshSizeMin decimal Minimum Mesh size (mm) used in archaeobotanical recovery
Notes varchar
RecoveryMethod1 varchar
RecoveryMethod2 varchar
RecoveryMethod3 varchar
RecoveryMethod4 varchar
Volume smallint Volume (litres) of the soil extracted for sampling.
