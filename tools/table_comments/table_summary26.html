StrontiumSequences
Where sequential sampling data are available, e.g. from laser ablation data on single teeth or sequential sampling (which may sometimes overlap) of hair or nails.
Column name Data type Column description
StrontSliceID varchar Primary key
StrontiumID smallint References the strontium table for sample that has been sequencesd (such as a tooth)
SequenceOrder tinyint Rank order of sequence (lower numbers are closer to root)
Datapoints/slice tinyint Number of datapoints per slice
SequenceTime tinyint Duration of reading in seconds
Material varchar Material on which analysis completed; e.g. dentine, enamel
TaxonCode varchar Species of the animal on which the analysis was completed
87Sr/86Sr decimal Strontium isotopic ratio (87Sr/86Sr)
2SE_87Sr/86Sr decimal Within-run precision of each analyses (2SE) (Iolite Version 2.5)
2SD_87Sr/86Sr decimal Propagated from external reproducibility (2SD) obtained from primary standard during the analytical session combined with the within-run precision of each analyses (2SE) (Iolite Version 2.5)
87Rb/86Sr decimal Ratio of 87Rb to 86Sr
2SE_87Rb/86Sr decimal Within-run precision of each analyses (2SE) (Iolite Version 2.5)
Mean_SrBeam decimal Average of total Sr beam intensity in V (sum of 84Sr, 86Sr, 87Sr, 88Sr)
84Sr/86Sr decimal Ratio of 84Sr/86Sr
2SE_84Sr/86Sr decimal Within-run precision of each analyses (2SE) (Iolite Version 2.5)
ppm tinyint Estimated Sr concentration
notes varchar
