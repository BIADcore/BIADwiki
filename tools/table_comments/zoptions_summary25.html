zoptions_TaxaList
The full taxonomic description of all species represented in the database with its associated Taxoncode 14C marked for mixed samples and aggregated taxon codes
Column name Data type Column description
TaxonCode varchar Unique code to identify the full taxonomic name. This usually consists of the first four letters of the genus and first three letters of the species. Case sensitive - animals are lowercase, plants are uppercase.
FullNameOfTaxon varchar Full taxonomic name
Species varchar Taxonomic rank of species
Genus varchar Taxonomic rank of genus
WildDomesticStatus enum Description of the wild/domestic status, including combination categories (wild & domestic) as well as inderterminate categories (wild/domestic); for plants: "domestic/wild" is an uncertain status; "domestic" is a taxon that is for sure domesticated; "wild" is a taxon that is for sure not domesticated; "cultivated" is a taxon that is wild but cultivated by humans
TaxonCategory enum Higher level common taxonomic classification e.g. 'amphibia', 'bat', 'bird'. Useful in analysis where only land mammals, for example, want to be included.
Class enum Taxonomic rank of class
Family varchar Taxonomic rank of family
Kingdom enum Taxonomic rank of phylum
